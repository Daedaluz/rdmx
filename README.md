# RDMX

```
DMX over UDP

Usage: rdmx [OPTIONS]

Options:
  -d, --device <DEVICE>      [default: /dev/ttyUSB0]
  -b, --bind <BIND>          [default: 0.0.0.0:1337]
  -m, --mode <MODE>          [default: termios2] [possible values: termios2, set-serial]
  -t, --throttle <THROTTLE>  [default: 45]
  -h, --help                 Print help
  -V, --version              Print version
```
